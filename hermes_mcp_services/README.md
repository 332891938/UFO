# Hermes MCP Services (Split Mode A)

This directory contains three standalone MCP services that can be copied and run independently:

- `linux_mcp` - Linux shell and system info MCP server
- `mobile_mcp` - Android ADB data/action MCP servers
- `windows_ui_mcp` - Windows desktop UI automation, CLI launch, PDF reading, and Office COM MCP server

Each service contains:
- `server.py`
- `requirements.txt`
- `README.md`

You can copy any subdirectory as-is to another machine and run it independently.
